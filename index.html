Welcome to HugeMCE

We're going to build HugeMCE, the 100% free fork of the world's #1 open source rich text editor, called TinyMCE.

But HugeMCE is a very new project, and it's not yet available on NPM or anywhere except GitHub. And it's not better than TinyMCE: We haven't made any changes yet except for files like README.md. Stay tuned though!
